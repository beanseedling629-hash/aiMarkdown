## 📋 项目概述

本项目是一个 **AI Markdown 插件**，旨在为用户提供强大的 Markdown 编辑和预览功能。本文档作为第二篇测试文章，使用更贴近实际场景的内容来测试插件的渲染效果。

### 核心功能

1. **实时预览** - 边写边看，即时渲染
2. **语法高亮** - 支持多种编程语言的代码高亮
3. **主题切换** - 支持明暗主题，保护眼睛
4. **导出功能** - 支持导出 HTML、PDF 等格式
5. **快捷键支持** - 提升编辑效率

## 📚 API 文档

### 核心类：`MarkdownParser`

用于解析 Markdown 文本并转换为 HTML。

#### 构造函数

```typescript
constructor(options?: ParserOptions)
```

**参数**：

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| options | `ParserOptions` | 否 | 解析器配置选项 |
| options.gfm | `boolean` | 否 | 是否启用 GitHub 风格 Markdown（默认 `true`） |
| options.breaks | `boolean` | 否 | 是否将换行转为 `<br>`（默认 `false`） |
| options.tables | `boolean` | 否 | 是否启用表格语法（默认 `true`） |

**示例**：

```typescript
import { MarkdownParser } from 'ai-markdown';

// 使用默认配置
const parser1 = new MarkdownParser();

// 自定义配置
const parser2 = new MarkdownParser({
    gfm: true,
    breaks: true,
    tables: true
});
```

#### 方法：`parse()`

```typescript
parse(markdown: string): string
```

将 Markdown 文本解析为 HTML 字符串。

**参数**：

- `markdown` (`string`): 要解析的 Markdown 文本

**返回值**：

- `string`: 解析后的 HTML 字符串

**示例**：

```typescript
const parser = new MarkdownParser();
const md = '# Hello\n\nThis is **bold**';
const html = parser.parse(md);
console.log(html);
// 输出: <h1>Hello</h1><p>This is <strong>bold</strong></p>
```

#### 方法：`parseInline()`

解析行内 Markdown 元素。

```typescript
parseInline(text: string): string
```

**示例**：

```typescript
const parser = new MarkdownParser();
const result = parser.parseInline('This is *italic* and this is `code`');
console.log(result);
// 输出: This is <em>italic</em> and this is <code>code</code>
```

## 🔧 高级用法

### 插件系统

AI Markdown 支持插件扩展，你可以编写自己的插件：

```typescript
import { Plugin } from 'ai-markdown';

class MyPlugin implements Plugin {
  name = 'my-plugin';
  version = '1.0.0';
  
  install(parser: MarkdownParser) {
    // 添加自定义语法
    parser.addSyntax({
      name: 'highlight',
      pattern: /==(.+?)==/g,
      replacer: (match, content) => {
        return `<mark>${content}</mark>`;
      }
    });
  }
}

// 使用插件
const parser = new MarkdownParser();
parser.use(new MyPlugin());
```

### 自定义渲染器

你可以自定义 HTML 渲染输出：

```typescript
const parser = new MarkdownParser({
  renderer: {
    heading(text, level) {
      return `<h${level} class="custom-heading">${text}</h${level}>`;
    },
    
    code(code, language) {
      return `<pre class="custom-code-block"><code class="language-${language}">${code}</code></pre>`;
    },
    
    table(header, body) {
      return `<div class="table-wrapper"><table class="custom-table">${header}${body}</table></div>`;
    }
  }
});
```

### Python 示例：文件转换工具

```python
#!/usr/bin/env python3
"""
Markdown 文件批量转换工具
支持转换为 HTML、PDF 等格式
"""

import os
import argparse
from pathlib import Path
from typing import List, Optional
import markdown
from weasyprint import HTML, CSS

class MarkdownConverter:
    """Markdown 转换器"""
    
    def __init__(self, input_dir: str, output_dir: str):
        self.input_dir = Path(input_dir)
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(parents=True, exist_ok=True)
        
    def convert_to_html(self, md_file: Path) -> Path:
        """转换为 HTML"""
        with open(md_file, 'r', encoding='utf-8') as f:
            md_content = f.read()
        
        html_content = markdown.markdown(
            md_content,
            extensions=['tables', 'fenced_code']
        )
        
        output_file = self.output_dir / f"{md_file.stem}.html"
        with open(output_file, 'w', encoding='utf-8') as f:
            f.write(f"""<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>{md_file.stem}</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
{html_content}
</body>
</html>""")
        
        return output_file
    
    def convert_to_pdf(self, md_file: Path) -> Path:
        """转换为 PDF"""
        html_file = self.convert_to_html(md_file)
        
        output_file = self.output_dir / f"{md_file.stem}.pdf"
        HTML(filename=str(html_file)).write_pdf(str(output_file))
        
        return output_file
    
    def batch_convert(self, format: str = 'html') -> List[Path]:
        """批量转换"""
        results = []
        for md_file in self.input_dir.glob('*.md'):
            if format == 'html':
                result = self.convert_to_html(md_file)
            elif format == 'pdf':
                result = self.convert_to_pdf(md_file)
            else:
                raise ValueError(f"不支持的格式: {format}")
            results.append(result)
            print(f"✓ 已转换: {md_file.name} -> {result.name}")
        
        return results

def main():
    parser = argparse.ArgumentParser(description='Markdown 批量转换工具')
    parser.add_argument('-i', '--input', required=True, help='输入目录')
    parser.add_argument('-o', '--output', required=True, help='输出目录')
    parser.add_argument('-f', '--format', choices=['html', 'pdf'], default='html', help='输出格式')
    
    args = parser.parse_args()
    
    converter = MarkdownConverter(args.input, args.output)
    converter.batch_convert(args.format)
    print(f"\n✨ 全部转换完成！")

if __name__ == '__main__':
    main()
```

### JavaScript 示例：实时预览编辑器

```javascript
/**
 * AI Markdown 实时预览编辑器
 * 使用 Virtual DOM 优化渲染性能
 */

class MarkdownEditor {
  constructor(container, options = {}) {
    this.container = typeof container === 'string' 
      ? document.querySelector(container) 
      : container;
    
    this.options = {
      theme: 'light',
      fontSize: 16,
      tabSize: 4,
      ...options
    };
    
    this.state = {
      content: '',
      cursorPosition: 0,
      scrollPosition: 0,
      isPreviewVisible: true
    };
    
    this.init();
  }
  
  init() {
    // 创建编辑器结构
    this.container.innerHTML = `
      <div class="markdown-editor ${this.options.theme}">
        <div class="toolbar">
          <button data-action="bold" title="粗体 (Ctrl+B)">B</button>
          <button data-action="italic" title="斜体 (Ctrl+I)">I</button>
          <button data-action="link" title="链接 (Ctrl+K)">🔗</button>
          <button data-action="code" title="代码 (Ctrl+`)">`</button>
          <span class="divider"></span>
          <button data-action="heading" title="标题">H</button>
          <button data-action="list" title="列表">≡</button>
          <button data-action="table" title="表格">⊞</button>
        </div>
        <div class="editor-container">
          <textarea class="editor-input" spellcheck="false"></textarea>
          <div class="editor-preview"></div>
        </div>
        <div class="status-bar">
          <span class="word-count">字数: 0</span>
          <span class="line-count">行数: 1</span>
          <span class="cursor-pos">列: 1</span>
        </div>
      </div>
    `;
    
    // 绑定事件
    this.bindEvents();
    
    // 应用主题
    this.applyTheme();
  }
  
  bindEvents() {
    const textarea = this.container.querySelector('.editor-input');
    const preview = this.container.querySelector('.editor-preview');
    const toolbar = this.container.querySelector('.toolbar');
    
    // 输入事件
    textarea.addEventListener('input', (e) => {
      this.state.content = e.target.value;
      this.updatePreview();
      this.updateStatusBar();
    });
    
    // 滚动同步
    textarea.addEventListener('scroll', () => {
      const percentage = textarea.scrollTop / (textarea.scrollHeight - textarea.clientHeight);
      preview.scrollTop = percentage * (preview.scrollHeight - preview.clientHeight);
    });
    
    // 工具栏点击
    toolbar.addEventListener('click', (e) => {
      const button = e.target.closest('button');
      if (button) {
        this.executeAction(button.dataset.action);
      }
    });
    
    // 快捷键
    textarea.addEventListener('keydown', (e) => {
      if (e.ctrlKey || e.metaKey) {
        switch (e.key) {
          case 'b':
            e.preventDefault();
            this.executeAction('bold');
            break;
          case 'i':
            e.preventDefault();
            this.executeAction('italic');
            break;
          case 'k':
            e.preventDefault();
            this.executeAction('link');
            break;
        }
      }
      
      // Tab 键支持
      if (e.key === 'Tab') {
        e.preventDefault();
        const start = textarea.selectionStart;
        const end = textarea.selectionEnd;
        const value = textarea.value;
        textarea.value = value.substring(0, start) + ' '.repeat(this.options.tabSize) + value.substring(end);
        textarea.selectionStart = textarea.selectionEnd = start + this.options.tabSize;
      }
    });
  }
  
  updatePreview() {
    const preview = this.container.querySelector('.editor-preview');
    // 使用虚拟 DOM 或直接将 Markdown 转为 HTML
    preview.innerHTML = this.parseMarkdown(this.state.content);
  }
  
  updateStatusBar() {
    const content = this.state.content;
    const words = content.trim().split(/\s+/).filter(w => w.length > 0).length;
    const lines = content.split('\n').length;
    
    this.container.querySelector('.word-count').textContent = `字数: ${words}`;
    this.container.querySelector('.line-count').textContent = `行数: ${lines}`;
  }
  
  executeAction(action) {
    const textarea = this.container.querySelector('.editor-input');
    const start = textarea.selectionStart;
    const end = textarea.selectionEnd;
    const selectedText = textarea.value.substring(start, end);
    
    let replacement = '';
    
    switch (action) {
      case 'bold':
        replacement = `**${selectedText || '粗体文本'}**`;
        break;
      case 'italic':
        replacement = `*${selectedText || '斜体文本'}*`;
        break;
      case 'link':
        replacement = `[${selectedText || '链接文本'}](https://)`;
        break;
      case 'code':
        replacement = `\`${selectedText || '代码'}\``;
        break;
      case 'heading':
        replacement = `# ${selectedText || '标题'}`;
        break;
      case 'list':
        replacement = `- ${selectedText || '列表项'}`;
        break;
      case 'table':
        replacement = `| 列1 | 列2 | 列3 |\n|------|------|------|\n| 内容 | 内容 | 内容 |`;
        break;
    }
    
    textarea.value = textarea.value.substring(0, start) + replacement + textarea.value.substring(end);
    textarea.selectionStart = start;
    textarea.selectionEnd = start + replacement.length;
    textarea.focus();
    
    this.state.content = textarea.value;
    this.updatePreview();
  }
  
  parseMarkdown(text) {
    // 简化的 Markdown 解析（实际应使用专业的解析库）
    let html = text;
    
    // 标题
    html = html.replace(/^### (.*$)/gim, '<h3>$1</h3>');
    html = html.replace(/^## (.*$)/gim, '<h2>$1</h2>');
    html = html.replace(/^# (.*$)/gim, '<h1>$1</h1>');
    
    // 粗体和斜体
    html = html.replace(/\*\*(.+?)\*\*/g, '<strong>$1</strong>');
    html = html.replace(/\*(.+?)\*/g, '<em>$1</em>');
    
    // 代码
    html = html.replace(/`(.+?)`/g, '<code>$1</code>');
    
    // 链接
    html = html.replace(/\[([^\]]+)\]\(([^)]+)\)/g, '<a href="$2">$1</a>');
    
    // 段落
    html = html.split('\n\n').map(para => {
      if (!para.match(/^<h[1-6]>|^<ul>|^<ol>|^<pre>|^<table>/)) {
        return `<p>${para}</p>`;
      }
      return para;
    }).join('\n');
    
    return html;
  }
  
  applyTheme() {
    this.container.querySelector('.markdown-editor').className = 
      `markdown-editor ${this.options.theme}`;
  }
  
  setContent(content) {
    this.state.content = content;
    this.container.querySelector('.editor-input').value = content;
    this.updatePreview();
  }
  
  getContent() {
    return this.state.content;
  }
}

// 使用示例
const editor = new MarkdownEditor('#editor', {
  theme: 'dark',
  fontSize: 16
});

editor.setContent(`# 欢迎使用 AI Markdown 编辑器

这是一个**实时预览**编辑器示例。

## 功能特性

- 实时预览
- 语法高亮
- 快捷键支持
- 主题切换

> 开始编辑吧！

\`\`\`javascript
console.log('Hello, World!');
\`\`\`
`);
```

## 🐛 常见问题

### 1. 代码高亮不生效

**问题描述**：代码块没有语法高亮。

**解决方案**：

- 检查是否正确指定了语言标识符
- 确认已安装语法高亮库（如 Prism.js 或 Highlight.js）
- 检查主题配置是否正确

```javascript
// 正确示例
const parser = new MarkdownParser({
  codeHighlight: 'prism'
});
```

### 2. 表格边框不显示

**问题描述**：表格没有边框。

**解决方案**：添加 CSS 样式

```css
table {
  border-collapse: collapse;
  width: 100%;
  margin: 1em 0;
}

th, td {
  border: 1px solid #e1e4e8;
  padding: 8px 16px;
  text-align: left;
}

th {
  background-color: #f6f8fa;
  font-weight: 600;
}
```

### 3. 图片无法显示

**问题描述**：图片路径正确但无法显示。

**可能原因**：

- 相对路径问题
- 文件权限问题
- 网络图片链接失效

**解决方案**：

```markdown
<!-- 使用绝对路径 -->
![图片](/assets/images/photo.png)

<!-- 使用 base64 编码（小图片） -->
![图片](data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mNk+M9QDwADhgGAWjR9awAAAABJRU5ErkJggg==)

<!-- 检查网络链接 -->
![图片](https://via.placeholder.com/150)
```
